macOS中的Emacs快捷键

发表于 2019-07-09 | 阅读次数:

自盘古开天辟地以来，世间流传着两件绝世神器，它们有着对文字强大的魔法、不同的风格以及各自虔诚的信徒，信徒们无时不刻对其拥护、吹嘘、争辩，为之后两大派系的奠定了基础。
这两件神器分别是，被称为编辑器之神Vim和神的编辑器Emacs。

这篇文章不涉及任何派系争斗，主要是讲解Emacs在macOS中快捷键的简单使用，掌握之后可以大大提升macOS中的文字编辑效率。有杠精会问，为什么不是Emacs而不是Vim，因为macOS原生在文本编辑方面绑定了Emacs的快捷键，所以macOS中的所有软件都进行了支持，比如weChat、notes、vsCode…下面进行一个简单步骤的讲解，了解之后还是需要多加练习，进入一个熟悉的过程，学成之后配合macOS一些基本快捷键便可以在90%的情境下抛弃鼠标，利用键盘完成自己想做的事。

1、对于非HHKB的键盘，第一步需要把键盘上的 Caps Lock 键改成 Ctrl, 这样键盘上将有三个Ctrl键，因为Emacs独特的快捷键组合，如果使用左下角的Ctrl，对于小拇指短的人来说是很不舒服的，所以把使用率极低的Caps Lock键改成Ctrl，小拇指可以很轻松的触上。在macOS中按如下操作。

2、步骤一完成之后，下面就是需要熟悉的方向键操作
Ctrl + P 上
Ctrl + N 下
Ctrl + B 左
Ctrl + F 右

然后是一些基础移动和修改键
Ctrl + A 移到句首
Ctrl + E 移到句尾
Ctrl + H 退格删除（向左删除）
Ctrl + D 向右删除
Ctrl + K 删除光标右侧所有文字
Ctrl + O 新增一行
CMD + Backspace 删除一整行
Ctrl + T 交换光标前的两个字符

最后是一些使用率比较高的操作系统快捷键
CMD + Tab 程序向右切换（按住CMD不放）
CMD + Shift + Tab 程序向左切换
CMD + Q 快速关闭程序
CMD + H 关闭当前窗口
CMD + M 缩小化当前窗口
CMD + Option + Esc 强制结束程序进程
CMD + ~ 快速切换同一程序下的不同窗口

以上快捷键虽少，但是足够日常使用，开始会比较难以适应，熟练之后对窗口和文字的管理便可游刃有余

更多快捷键可以参考macOS全快捷键大全
